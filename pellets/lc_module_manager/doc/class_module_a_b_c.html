模块实现示例

模块示例类。
模块定义必须包含在BEGIN_DECLARE_MODULE与END_DECLARE_MODULE两个宏之间； 编写定义前必须使用宏IMPLEMENT_MODULE_BASE对该模块类进行定义；
模块必须按单例模式实现，实现格式必须与此类一致。
